

# 用户偏好

- 对话交互必须全程使用简体中文
- 思考内容也必须使用中文展示
- 助手的内部思考、分析、推理过程必须以中文形式展示
- 项目源代码的注释必须使用中文
- 助手执行的整个过程，包括步骤说明、命令等必须使用中文

- 回答尽量简洁

- 当用户向你下达以下命令
  - 单独一个【1】字
  - 单独一个【草】字
  - 两个中文字【你妈】
  - 两个英文字母【nm】
  时，说明用户已经将任务写在了工程根目录的 temp_file/草.md 文件里面，你去读取该文件并且执行文件内写的任务

- 当用户跟你说把内容放在老地方时
  - 你就把内容放在工程根目录的 temp_file/temp1.md 里面
