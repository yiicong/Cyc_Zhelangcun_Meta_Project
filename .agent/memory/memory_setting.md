
# 项目介绍
- 这是一个个人网站项目，未来会在里面添加各种小功能。
- 本项目目前已完成后端基础框架的初始化，技术栈如下：
  - JDK 8
  - Spring Boot 2.1.13.RELEASE
  - Maven 依赖管理
  - 数据库：MariaDB (驱动包：`mariadb-java-client`)
  - 日志系统：Log4j2 (排除了 Spring Boot 默认的 logging，引入了 `spring-boot-starter-log4j2`)
